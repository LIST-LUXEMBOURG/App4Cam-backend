# PolliCAM Backend

## Setup

1. Run: `npm install`
2. Copy config file `.env.sample` as `.env`.
3. Edit config file `.env` as needed.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
